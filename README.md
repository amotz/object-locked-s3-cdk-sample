# Object Locked S3 CDK Sample

Building S3 (with object lock) using AWS CDK.

## Usage

### Install

```bash
npm install -g aws-cdk
npm install
```

### Build & Deploy

```bash
npm run build
cdk bootstrap
cdk deploy
```
